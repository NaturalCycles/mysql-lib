# Develop

Install MySQL server+client on Mac:

```shell

brew install mysql

# start

mysql.server start

# secure

mysql_secure_installation

# connect

mysql -uroot

# Default options are read from the following files in the given order:
# /etc/my.cnf /etc/mysql/my.cnf /opt/homebrew/etc/my.cnf ~/.my.cnf

```
